# AlgoPay semifinal judging materials

Everything you need for the semifinal round (3 min presentation + 2 min Q&A per table).

## Quick start

1. Paste slide text from [`SLIDE_CONTENT.md`](SLIDE_CONTENT.md) into the [Algorand template](../Copy%20of%20Algorand-Presentation-Template%20(1).pptx) — **do not change branding**.
2. Embed [`architecture.svg`](architecture.svg) on Slide 2.
3. Practice [`PRESENTATION_SCRIPT.md`](PRESENTATION_SCRIPT.md) with a timer.
4. Rehearse demo via [`LIVE_DEMO_RUNBOOK.md`](LIVE_DEMO_RUNBOOK.md).
5. Memorize [`PANEL_PREP_CHEATSHEET.md`](PANEL_PREP_CHEATSHEET.md).
6. Send [`VALIDATION_KIT.md`](VALIDATION_KIT.md) outreach; fill feedback log with real quotes.
7. Read monetization model: [`docs/ecosystem/MONETIZATION.md`](../../docs/ecosystem/MONETIZATION.md).
