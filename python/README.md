# Python SDK (`algopay-sdk`)

AlgoPay client for AI-agent payments on Algorand: wallets, USDC (ASA), x402, guards, ledger.

Install from repository root:

```bash
pip install -e "./python[dev]"
```

Imports use **`algopay`**. Canonical project metadata lives in **`pyproject.toml`** in this folder. Examples: **`examples/`**. Tests: **`tests/`**.
